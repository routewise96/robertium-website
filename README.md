# Robertium Website

Landing page for [Robertium](https://robertium.com) — open-source infrastructure for literature-based drug discovery.

## Stack

- Static HTML/CSS, no framework
- Deployed on Cloudflare Pages
- CI/CD via GitHub Actions

## Local development

Open `index.html` in a browser. That's it.

## Deployment

Pushes to `main` trigger automatic deployment to Cloudflare Pages.
